# 🌊 VibeFlow - Social Media Platform

VibeFlow is a modern MERN stack social media application designed for seamless connection and vibrant community interaction.

## 🚀 Features

- **Authentication**: Secure Login/Signup using JWT and Context API.
- **Dynamic UI**: Fully responsive design built with Vite and MUI/Tailwind.
- **Real-time Updates**: (Add more features as you build them!)

## 🛠️ Tech Stack

- **Frontend**: React (Vite), Material UI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB

## 📦 Installation

1. Clone the repo: `git clone https://github.com/your-username/VibeFlow.git`
2. Install dependencies: `npm install`
3. Create a `.env` file (see below).
4. Start the app: `npm run dev`

## 🔑 Environment Variables

To run this project, you will need to add the following variables to your `.env` file:
`VITE_API_URL`
`VITE_JWT_SECRET`
